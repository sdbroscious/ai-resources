# State of AI Coding Tools (at the CLI)
*2026-01-29*

> This is a summary of my experience using Claude Code, GitHub Copilot CLI, and Gemini CLI. Casey D asked for it. The original is stored in [my Google Drive](https://docs.google.com/document/d/1ormm_mp01j4-CHXX1ccqgeltezStTFiJMQxV14f-ct8/edit?tab=t.0#heading=h.esflwaacpw8b).

I’ve used Claude Code, GitHub Copilot CLI, and Gemini CLI. I see the three like this:

* **Claude Code is like an Acura RX**: It comes with a high price but performs extremely well and has first class features, particularly for car aficionados or those who love luxury  
* **GitHub Copilot CLI is like a Honda Accord**: It performs reasonably well in most scenarios and has a reasonable price tag, making it a sensible choice for most  
* **Google Gemini CLI is like a Honda Civic**: It’s economically priced but lacks the performance and features of an Accord or RX, making it attractive to the budget conscious

In addition to being a solid, sensible choice at the tool level, GitHub Copilot has several features that make it attractive in an enterprise setting:

* **Governance**: Copilot is a fully integrated part of the GitHub universe and so takes advantage of centralized provisioning and enablement of users, policies and features.  
* **Predictable spend**: Copilot licensing is based on a flat rate of premium requests (per user per month). Claude Code pricing is token based and thus more volatile.  
* **Trusted partner**: GitHub and Microsoft are already trusted partners of Great American Insurance. In addition, they are both established, healthy companies. Anthropic, by contrast, is a promising startup but still a startup that hasn’t been fully tested by time.  
* **Flexibility without vendor lock-in**: Copilot offers frontier models from a variety of leading AI labs (Anthropic, Microsoft, OpenAI, Grok) whereas Claude Code is limited to Anthropic’s models.

Jim Kohl and I attended GutHub’s developer conference in October 2025\. I came away from the conference convinced that Microsoft and GitHub are committed to ensuring that Copilot remains a top-tier enterprise AI coding tool that fits tightly into the larger GitHub landscape. 

That said, we need to preserve our investment in GitHub Copilot and continue to monitor Copilot’s roadmap and the product’s actual evolution to verify that it remains an Accord and doesn’t become a Civic.

We should also keep an eye on Claude Code’s evolution to see whether it grows into a true enterprise tool and whether its feature set evolution outpaces Copilot. It may make sense to have a select subset of developers equipped with Claude Code licenses to do this.
